# ML Projects

A collection of end-to-end Machine Learning projects built using Python and scikit-learn.

This repository showcases practical implementations of machine learning algorithms on real-world datasets. Each project follows a complete machine learning workflow, from data preprocessing to model evaluation.

As I continue learning, this repository will grow with additional projects covering both regression and classification problems.

---

##  Machine Learning Workflow

Each project follows a structured pipeline:

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Train-Test Split
6. Feature Scaling (when required)
7. Model Training
8. Model Evaluation
9. Prediction
